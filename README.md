# Вычислительная ниткография

Программа строит схему нитей для портрета из ниток по фотографии. Алгоритм основан на методе Filtered Back Projection из рентгеновской томографии.

## Как работает

1. **Загружаем фото** — нужно квадратное, чёрно-белое. Программа проверяет формат и размеры.
2. **Подготавливаем** — поднимаем яркость и инвертируем, чтобы тёмные детали (глаза, волосы) получили больше нитей.
3. **Преобразование Радона** — для каждого угла считаем проекции — интегралы вдоль прямых. Получаем синограмму.
4. **Рамп-фильтр** — сворачиваем каждую проекцию с фильтром по формуле из задания.
5. **Прореживание** — из всех возможных прямых оставляем только самые яркие.
6. **Нормировка** — отрицательные значения зануляем, остальные приводим к диапазону [0, 1].
7. **Случайный дизеринг** — превращаем вероятности в бинарную схему: с вероятностью *p* протягиваем нить, иначе нет.
8. **Результат** — CSV координаты + PNG/SVG.

## Запуск

```bash
cd fbp_solution
bash run.sh
```

Или вручную:

```bash
cd fbp_solution
python3 build_schema.py input.png --angles 360 --threads 40 --out out
```

Результат появится в папке `out/`: схема нитей, превью, промежуточные изображения.

## Настройки


| Параметр            | Дефолтно | Что делает                                                                         |
| ------------------- | -------- | ---------------------------------------------------------------------------------- |
| `--angles`          | 360      | Число углов. Больше — точнее, но медленнее                                         |
| `--threads`         | 40       | Сколько нитей на каждый угол                                                       |
| `--brightness-lift` | 0.2      | Подъём яркости фото. Нужен, чтобы после фильтра было меньше отрицательных значений |
| `--dither-mode`     | random   | Или `error-diffusion` — чуть лучше качество                                        |
| `--prob-gamma`      | 0.85     | Управляет контрастом вероятностей                                                  |
